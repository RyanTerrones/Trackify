Trackify

A music discovery Progressive Web App built with Ionic and Angular.

Description
Trackify allows users to search, discover and explore music using the Last.fm and Deezer APIs. Features include a mini music player, favourites, search history, charts, genre discovery and multi-language support.

Live Demo
https://trackify-c44ec.web.app

Tech Stack
- Ionic 7 / Angular (Standalone Components)
- Last.fm API
- Deezer API
- IonicStorage
- Capacitor Plugins (Share, Clipboard, Local Notifications, Network)
- Firebase Hosting

Running Locally
npm install 
ionic serve
